
Hola chicas,

Comparto con vosotros esta carpeta para que Sonia pueda empezar a añadir sus experimentos :)

* En esta carpeta se encuentra el template para crear nuevos experimentos
* Para crear un nuevo experimento copia el fichero template.md y crea uno nuevo que se llame con el identificador único del experimento, guión bajo, el idioma y la misma extensión. Esto es, si es el primer experimento, se llamará "A000001_spanish.md", si es el segundo "A000002_spanish.md".  Si es el tercer experimento y lo creas en inglés sería "A000003_english.md"
* Utiliza un editor de texto plano (por ejemplo, el block de notas de Windows) para modificar este fichero. Si tienes ganas, búscate un editor de Markdown (que es el formato de texto plano utilizado) para poder ver más bonito el texto mientras lo escribes.
* Recuerda que cuando guardes los cambios el fichero debe guardarse en la misma carpeta. Si se incluye alguna imagen (de cabecera o "main" o de contenido o "content" deberá incluirse en la subcarpeta "img")
* Más abajo verás una explicación sobre lo que va en cada apartado, aunque el fichero template.md es autoexplicativo.
* Escribas los experimentos en inglés o en español, no modifiques los textos:
 Title, id, level, fields, tags, language, main-image, content-image, Abstract, Materials, Experiment, Easy Explanation, Scientific Explanation, Bibliography.
* Estos textos se utilizarán como códigos para convertir de forma automática tu contenido en algo visual ;)


Explicación de los campos:


- Título
- Identificador numérico (así cada uno de los experimentos tiene un id que podemos usar como referencia dentro del sistema)
- Áreas de conocimiento (separadas por comas si son más de una): química, biología
- Nivel de dificultad: un valor numeric del 1 al 10 que podamos correlacionar con intervalos de edad
- Idioma: spanish/english
- tags: etiquetas separadas por comas para facilitar búsquedas, relacionar experimentos entre sí, etc...
- Imageness: caratula y content
- Descripción Breve (como un abstract)
- Materiales necesarios
- Experimento (descripción detallada, pasos a seguir, etc...)
- Explicación científica
- Explicación sencilla
- Bibliografía complementaria
